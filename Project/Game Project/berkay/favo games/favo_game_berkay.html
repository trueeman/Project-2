Mijn meest favoriete game is een spel genaamd Ark.
Ark is een PVP survival game met veel verschillende dingen die je kan doen zoals: dinosaurussen temmen, wapens maken, karakters customisen, een basis bouwen en andermans basis binnenvallen
Mijn 2de favoriete game is DayZ.
Dayz is een beetje het zelf als Ark maar i.p.v dinosaurussen zijn er zombies die je de hele tijd aanvallen. In DayZ kan je geen zombies temmen maar je moet ze doodmaken om spullen te krijgen zoals: eten, wapens, resources en kogels.
Als laatst hebben we de game genaamd Gta V.
Gta V is een spel waar je een story mode en online mode hebt. In de story mode heb je veel manieren om vals te spelen. Maar als je de story mode normaal speelt kun je veel lol hebben. In GTA V online kan je mensen irriteren en heists starten.
