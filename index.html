Ik heb geen favoriete game maar ik heb een favoriete serie aan games (games met meerdere delen). De game heet Yakuza
Yakuza is een serie van games die gaan over het leven van een Yakuza (japanse mafia lid).
Dit is mijn favoriete game, omdat ik de verhalen heel goed vind. Er is ook veel actie en spanning.
Mijn 2de favoriete game is Friday the 13th.
Friday the 13th is een horrosspel waarbij je moet verstoppen van Jason Vorhees (moordenaar met een hocky masker).
Dit is mijn favoriete game omdat het een spannende game is die je ook samen kan spelen. Ik speel dit spel daarom ook altijd met een vriend van mij.
Als laatste heb je de game Call of duty black ops 2.
Cod 2 is een fps (first person shooter game) waarbij je online tegen andere mensen speelt.
Dit is mijn favoriete game omdat het vooral nostalgisch is. daarnaast heb je ook andere gamemodes bijvoorbeeld (zombies).
